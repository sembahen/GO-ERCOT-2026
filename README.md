# GO-ERCOT-2026

Updated GO EERCOT workflow for simulation and analysis of the Electric Reliability Council of Texas (ERCOT) grid operations.

## Overview

This project contains a Python-based workflow for simulating and analyzing ERCOT grid operations. It processes inputs, runs simulations, and generates outputs for analysis and reporting.

## Project Structure

```
GO-ERCOT-2026/
├── inputs/              # Input data and configuration files
├── outputs/             # Generated output files and results
├── simulation_folders/  # Simulation-related files and scripts
└── README.md           # This file
```

### Directories

- **inputs/**: Contains input data and configuration files needed for simulations
- **outputs/**: Stores all generated output files and results from simulations
- **simulation_folders/**: Contains simulation-related scripts and supporting files
